algebra.nvim
------------

Computer algebra system for Neovim.

| Input | Output |
|-------|--------|
| `1 + 1` | `2` |
| `2 * 3 + 1` | `7` |
| `x + x` | `(2 * x)` |
| `(x + 1)^2` | `((x ^ 2) + ((2 * x) + 1))` |
| `(x + 1) * (x - 1)` | `((x ^ 2) + -1)` |
| `(x + 1)^5` | `((x ^ 5) + ((10 * (x ^ 3)) + ((10 * (x ^ 2)) + ((5 * (x ^ 4)) + ((5 * x) + 1)))))` |
| `(x - 1)^2` | `((x ^ 2) + ((-(2 * x)) + 1))` |
